# Портфолио видеомонтажёра

## Что заменить

1. В `index.html` замените имя, описание, статистику, проекты и контакты.
2. Положите своё видео в `assets/showreel.mp4`. Лучше экспортировать MP4, H.264, Full HD; для быстрой загрузки желательно до 50–100 МБ.
3. Замените `assets/showreel-poster.svg` на свой постер, если нужно. Допустимы JPG, PNG и WebP — тогда измените имя в атрибуте `poster`.
4. Для связи на сайте уже указаны Telegram `https://t.me/vneskila322` и почта `vneskila322@gmail.com`.

## Как посмотреть

Можно дважды открыть `index.html`. Для полноценной проверки лучше запустить в папке простой локальный сервер, например: `python -m http.server 8000`, затем открыть `http://localhost:8000`.

## Как опубликовать бесплатно

### GitHub Pages

1. Создайте новый публичный репозиторий на GitHub.
2. Загрузите в корень репозитория содержимое этой папки.
3. Откройте Settings → Pages.
4. В разделе Build and deployment выберите Deploy from a branch, ветку `main` и папку `/ (root)`.
5. Сохраните. Через несколько минут GitHub покажет адрес сайта.

### Netlify

1. Откройте app.netlify.com и войдите.
2. Выберите Add new site → Deploy manually.
3. Перетащите всю папку `purple-video-portfolio` в окно загрузки.
4. Сайт сразу получит публичную ссылку; домен можно изменить в настройках.

Для тяжёлого шоурила удобнее загрузить видео на Vimeo или YouTube и вставить iframe: так сайт будет открываться быстрее и не упрётся в ограничения хостинга.
